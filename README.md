## Getting Started

To get a local copy just follow these simple example steps.

### Prerequisites


* A Code Editor of your choice
  ```sh
  I use VSCode
  ```

### Installation


1. Clone the repo
   ```sh
   git clone https://github.com/StevenPss/kaizerchiefs-redesign
   ```
2. Change directory
   ```sh
   cd kaizerchiefs-redesign
   ```
3. Install dependencies
   ```sh
   npm i
   ```
